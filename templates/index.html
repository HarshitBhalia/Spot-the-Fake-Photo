System Architecture

This detector utilizes a 4-signal hybrid pipeline to measure both handcrafted physical anomalies and deep visual cues:

  • FFT / Moiré: Extracts frequency-domain features to catch high-frequency interference patterns caused by screen pixel grids.
  • Specular Glare: Detects blown-out, low-saturation highlights typical of ambient light reflecting off glossy screen glass.
  • Bezel Detection: Uses Canny + Hough transforms to identify straight edges near image borders indicating phone/laptop bezels.
  • MobileNetV2: A fine-tuned CNN head processes high-level textures and color-shift anomalies.

These 19 features are fused via a Logistic Regression meta-classifier.
